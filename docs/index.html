Projects

IMDb Top 250 Movie Analysis

This project explores IMDb's Top 250 movies to uncover patterns in genre performance and rating trends across decades. Using Google Sheets, pivot tables and visualizations, I analyzed which genres dominate among the top films and how the average movie rating and audience interests have evolved over time. The project demonstrates data cleaning, aggregation, and insight generation from non-random datasets.

Sleep Health Analysis

This project explores how sleep quality, physical activity, and occupation relate to overall health outcomes such as BMI, blood pressure, and stress levels. Using a dataset from Kaggle, I performed data cleaning, transformation, and visualisation to identify trends and patterns that may connect sleep behaviour with health and occupational factors.

I imported the Kaggle dataset to Google Sheets and performed most of the data cleaning and transformation there. I then imported the cleaned data into Looker Studio to create the visualisations.
